# dns-wizard/src-tauri

## Purpose

Tauri Rust backend for the DNS wizard desktop app.

## Ownership

Child of `dns-wizard/`. Cargo-based Rust project.

## Local Contracts

- Rust project via `Cargo.toml`. Not a JS project.
- Tauri configuration in `tauri.conf.json`.
- Uses Tauri v2 APIs.

## Work Guidance

- Check: `cargo check`
- Build: `cargo build`
- Format: `cargo fmt`
- Lint: `cargo clippy`
